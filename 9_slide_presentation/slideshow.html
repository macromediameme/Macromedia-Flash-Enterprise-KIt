The slideshow tag prints out a list of bullet points withing two colored bars. The tag has the standard width and height attributes, as well as attributes that control the color of the top bar; COLOR for text, barcolor for the bar, buttoncolor for the interface normal state, and mouseovercolor for the color when the mouse is hovering. It also takes the title of the presentation as well as the page that you are starting on.

Inside the tag are groups of items that are printed out. Up to six on a page. Each item consists of a bullet and a bit of text that link to a web page. Color and Bulletcolor control the appearance of an item, while href and target mimic the HTML anchor tag. The text is set with the text attribute, while it's order is set by the page attribute.
