This is an example of the <MMflash:newsHeadlines> tag. It takes two arguments. A speed and a source. The source can be any XML file that is in the iSyndicate XML format.

<MMflash:newsHeadlines src="news.xml" speed="2" target="contentFrame"/>

It can only load xml files from the same TLD unfortunatly. Try setting the src to entertainment.xml or technology.xml and the headlines will change.

The speed attribute determines how many pixels per frame it will scroll.

The target determines the window target that the links will open into.

Width and height attributes set the width and height.

You can change headlines by calling the changeHeadlines() method. Change it to the entertainment xml file.

The colors can be set and modified interactively by setting the properties "color", "active" and "hover".

The target can be changed with the changeTarget() method.
